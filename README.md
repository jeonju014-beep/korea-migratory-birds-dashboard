# 🌸 우리 동네 철새 분포 대시보드

대한민국 겨울철새 분포 현황과 **댕기머리물떼새(Northern Lapwing)** 서식·이동 경로를 보여주는 대시보드입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 **http://127.0.0.1:5173/** 로 접속하세요.

## 포함 내용

- 2024–2025 겨울철새 동시센서스 요약
- 전국 주요 철새 도래지 지도
- 대표 겨울철새 소개 카드
- 댕기머리물떼새 사진, 국내 서식지 지도, 이동 경로 지도

## 기술

React · TypeScript · Vite · Tailwind CSS · Leaflet
